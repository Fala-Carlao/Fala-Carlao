## Sobre mim

Trabalho com **operações de NOC** em um provedor de internet (fibra/FTTH), lidando diariamente com monitoramento de rede, infraestrutura e resolução de incidentes.

Sou apaixonado e estudante constante de **automações, redes de computadores, infraestrutura, hardware e sistemas**. Fora do trabalho, mantenho um homelab self-hosted (**Central Usguri**) onde rodo todos os meus projetos pessoais — do banco de dados ao túnel de acesso externo, tudo containerizado e documentado.

Minha filosofia: entender o sistema de ponta a ponta. Se algo pode ser automatizado, eu automatizo.

## Como eu trabalho

- 🏠 **Homelab first** — tudo self-hosted, do banco ao proxy
- 📝 **Documentação em Obsidian** — decisões, arquitetura e runbooks registrados
- 🔄 **Automação sempre** — backups com cron, relatórios via n8n, deploys com compose
- 🛡️ **Segurança levada a sério** — hash de senha forte, CSRF, rate limit, secrets fora do repo
